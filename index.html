Interactive Accident Analysis Dashboard

This dashboard visualises road traffic accident data using interactive charts and maps. Users can examine accident trends, severity levels, hotspot locations, and contributing factors such as time, weather, and road conditions. The dashboard supports exploratory data analysis and improves understanding of road safety risks.
